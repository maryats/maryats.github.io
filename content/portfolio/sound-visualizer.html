This was one of my first ever coding projects, created for the Programming Basics course at Northeastern University. I wanted to create an interactive sound visualizer that responds to audio and user input.

>> View on GitHub

The mouse-following particle effects use a modified version of the particle effects are easy script by Hunter Loftis, and the particle size is tied to the volume of the sound captured from the browser microphone (using the p5.js sound library).
